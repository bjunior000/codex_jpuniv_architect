# 요코하마국립대 건축학과 외국인 유학생 입시 정리

기준 지원 단위: `도시과학부 건축학과(都市科学部 建築学科), YGEP-N1`

## 1. 최신성 판단

- 2026-03-24 기준, `2026년 4월 입학자 대상` 공식 개요가 최신으로 확인됐다.
- YNU는 학부 사비 외국인 유학생 입시를 `YGEP-N1`으로 운영한다.

## 2. 모집 인원

- 도시과학부 건축학과: `2명`
- 지원 가능 방식: `도일입시(渡日入試)`와 `도일 전 입시(渡日前入試)` 모두 가능

## 3. 일정

- 모집요강/원서 공개 시기: `2025년 9월 상순`
- 출원 기간:
  - 도일입시: `2025-10-30 ~ 2025-11-05`
  - 도일 전 입시: `2025-10-30 ~ 2025-11-05`
- 도일입시 시험일: `2025-12-20`
- 도일 전 입시: `서류 + 일본어 비디오레터 + 인터넷 면접`

## 4. EJU 요구 과목

- 일본어
- 이과: `물리 + 나머지 1과목`
- 수학: `코스2`

## 5. 영어 점수

- `TOEFL iBT`, `TOEIC L&R`, `IELTS Academic` 중 하나가 필요하다.
- 확인된 개요 페이지에서는 `최저 점수 컷`은 직접 보이지 않았다.
- 도시과학부는 영어 성적을 `중시`한다고 안내한다.

## 6. 추가 요구사항

- `일본어능력시험 N1 상당(JLPT N1相当)`을 전제로 하지만, JLPT 시험 자체는 필수는 아니다.
- 도일 전 입시는 `고교 성적표 + EJU + 영어 성적 + 일본어 비디오레터 + 온라인 면접`을 쓴다.

## 7. 공식 출처

- YGEP-N1 학부 입시 개요
  - https://global.ynu.ac.jp/admissions/college_outline/
- YNU 외국인 학부 입시 안내
  - https://www.ynu.ac.jp/international/accept/exam/faculty/index.html
- 건축학과 입학자 수용 방침
  - https://www.ynu.ac.jp/education/plan/initiative/admission/us_dept3/

## 8. 따로 준비해야 하는 항목

- `도일 전 입시(渡日前入試)` 지원자는 `일본어 비디오레터`를 따로 준비해야 한다.
- 온라인 면접도 있으므로, 일본어로 자기소개·지원동기·학업계획을 말하는 준비가 필요하다.
- 공식 개요 페이지 기준으로 `자기소개서`나 `포트폴리오` 명칭은 직접 확인되지 않았다.
