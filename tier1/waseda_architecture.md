# 와세다대 건축학과 외국학생입시 정리

기준 지원 단위: `창조이공학부 건축학과(創造理工学部 建築学科)`

## 1. 최신성 판단

- 2026-03-24 기준, 와세다는 `2027년도 외국학생을 위한 학부 입학시험 개요(2027年度 外国学生のための学部入学試験 試験概要)`를 이미 공개했다.
- 따라서 이 문서는 `2027년 4월 입학 기준 최신 공개 정보`를 우선 사용했다.

## 2. 모집 인원

- 외국학생입시 전용 모집 인원은 이번 확인 범위의 공식 개요 페이지에서 직접 확인되지 않았다.

## 3. 일정

- 출원 기간: `2026-06-01 ~ 2026-06-18`
- 1차: 서류전형
- 1차 합격 발표: `2026-09-15`
- 2차(건축학과): `면접시험 + 공간표현(필기시험)` `2026-10-17`
- 최종 합격 발표: `2026-10-26`
- 1차 입학 수속: `2026-10-26 ~ 2026-11-13`
- 2차 입학 수속: `2026-10-26 ~ 2026-11-27`

## 4. 추가 요구사항

- 건축학과는 `공간표현(空間表現)` 시험이 들어간다.
- `자기PR 자료(自己PR資料)`를 바탕으로 한 발표·면접이 중요하다.

## 5. 영어 점수

- 이번에 확인한 공식 외국학생입시 개요 페이지에서는 `TOEFL / TOEIC / IELTS 제출 의무`가 직접 확인되지 않았다.
- 학과 소개 페이지에서는 건축학과가 `일본어 기반 과정(Japanese-based Program only)`이라고 명시돼 있다.

## 6. 공식 출처

- 와세다 이공학술원 입시요항 페이지
  - https://www.waseda.jp/fsci/admissions_us/application/
- 건축학과 소개 페이지
  - https://www.waseda.jp/fsci/en/about/departments/creative/
- 일반선발 특설 사이트
  - https://admission.waseda.jp/

## 7. 따로 준비해야 하는 항목

- `자기PR 자료(自己PR資料)` 준비가 사실상 필수다.
- 공식 안내에 따르면, 작품, 연구, 조사, 사회공헌 활동 등 자신을 보여줄 자료를 자유 형식으로 준비해야 한다.
- `공간표현(空間表現)` 필기시험 대비도 별도로 필요하다.
- 즉, 와세다는 서류만이 아니라 `자기소개 자료 + 표현 시험 준비`가 큰 비중을 차지한다.
