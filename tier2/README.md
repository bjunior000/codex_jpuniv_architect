- 메이지대

- 릿쿄대
- 추오대
- 호세이대(도쿄)
- 간사이대
- 간사이가쿠인대
- 도시샤대
- 리츠메이칸대
- 규슈대
- 홋카이도대
- 니혼대

파일 구성:

- `*_architecture.md`: 학교별 요강 정리
- `*_architecture_translation_ko.md`: 공식 PDF/공식 안내 기준 번역 정리본

현재 분류 결과:

- 건축계열 지원 가능 확인: 메이지, 호세이, 간사이, 간사이가쿠인, 리츠메이칸, 규슈, 홋카이도, 니혼
- 건축 인접 전공으로 대체 정리: 추오(도시환경학과)
- 학부 기준 건축계열 모집 단위 부재 확인: 아오야마가쿠인, 릿쿄, 도시샤

표기 원칙:

- 일본어 또는 한자만으로 된 용어는 가능한 한 `한국어(일본어 원문)`으로 병기
- 일반적인 입시 용어는 한국어로만 표기
