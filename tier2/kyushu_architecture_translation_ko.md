# 규슈대 건축학과 모집요강 번역 정리본

대상 원문:

- `2026년도 사비 외국인 유학생 입시요강(令和8年度 私費外国人留学生入試 学生募集要項)`
- 공식 PDF: https://www.kyushu-u.ac.jp/f/63057/R8_shihi_inst.pdf

주의:

- 이 문서는 `공학부 건축학과(工学部 建築学科)` 지원자 기준으로 필요한 부분만 한국어로 재구성한 번역 정리본이다.

## 1. 모집 단위와 인원

- 공학부 건축학과는 외국인 유학생 입시 모집 대상이다.
- 모집 인원은 `약간명`이다.

## 2. 일정

- Web 입력/검정료 납부: `2025-10-27 ~ 2025-11-07`
- 출원 기간: `2025-11-04 ~ 2025-11-14 17:00 필착`
- 수험표 발행: `2026-01-30경`
- 건축학과 면접: `2026-02-27 11:00부터`
- 합격 발표: `2026-03-03`

## 3. EJU 요건

공학부 기준 번역:

- 일본어: 필요
- 수학: `코스2`
- 이과: `물리 필수 + 화학/생물 중 1과목`

출원 기준점:

- 일본어(독해·청해·청독해): `300점 이상`
- 수학, 이과는 `기준점 없음`

## 4. 영어 요건

- 제출 가능한 시험:
  - `TOEFL iBT`
  - `IELTS Academic`
  - `IELTS Online`(농학부 제외)
  - `케임브리지 영어 검정`
- 유효 기간: `2024년 1월 이후`
- 최소점은 건축학과 기준으로 직접 제시되지 않는다.

## 5. 선발 방식

- 공학부는 본교 일본어 시험 대상이 아니다.
- `영어 성적 + EJU + 출원서류 + 면접`으로 판정한다.
- 면접에서는 기초학력 확인이 포함될 수 있다.
- 발표 자료 반입은 불가하다.

## 6. 준비 서류

- `신상기록서(身上記録書)` 제출 필요
- 공학부는 `지원이유서` 제출 대상이 아니다.
