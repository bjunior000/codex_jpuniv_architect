# 메이지대 건축학과 모집요강 번역 정리본

대상 원문:

- `2027년도 외국인 유학생 입학시험 시험과목·일정표(2027年度外国人留学生入学試験 試験科目・日程等一覧)`
- `2026년도 외국인 유학생 입학시험 요강(2026年度 外国人留学生入学試験要項)`

주의:

- 2027은 현재 `요약 일정표`만 공개되어 있고 상세 모집요강은 아직 없다.
- 그래서 이 문서는 `2027 최신 일정표 + 2026 상세 PDF`를 결합한 번역 정리본이다.

## 1. 어떤 전형인가

- 메이지대 학부 외국인 유학생 입시는
  - `I형(Ⅰ型)`: 메이지대에서 시험을 보는 방식
  - `II형(Ⅱ型)`: 서류전형만으로 보는 방식
- 건축학과는 `이공학부 건축학과(理工学部 建築学科)` 소속이며 `I형` 대상이다.

## 2. 모집 단위와 정원

원문 해석:

- 이 전형에서 이공학부 전체 모집 인원은 `20명`
- 건축학과, 기계, 화학, 정보 등 여러 학과가 이 정원을 공유한다

즉:

- 건축학과 단독 정원이 아니라 `이공학부 통합 정원` 구조다.

## 3. 출원 자격

원문을 풀면 다음과 같다.

- 일본 국적이 아닌 사람
- 외국 12년 과정 이수(예정) 또는 그에 준하는 자격 보유
- `EJU`를 응시했어야 함
- 각 학부가 지정한 `영어 외부시험` 기준을 충족해야 함

## 4. 건축학과 EJU 요건

2027 최신 일정표 기준 번역:

- 일본어
- 이과 `물리 + 화학`
- 수학 `코스2`
- 합계 `4과목`

유효한 EJU 회차:

- `2025년도 제1회`
- `2025년도 제2회`
- `2026년도 제1회`

확인 포인트:

- 현재 공개된 2027 일정표에서는 `최저점`은 보이지 않는다.
- 즉 `과목 지정형`이지, `공개된 점수 컷형`은 아니다.

## 5. 영어 시험

원문 번역:

- 제출 가능한 시험은 `TOEFL iBT` 또는 `TOEIC L&R`
- 건축학과는 `출원 기준점 없음`
- 유효 수험 기간은 `2024-10-01 ~ 2026-07-31`

2026 상세 요강의 세부 번역:

- TOEFL:
  - ETS에서 학교로 직접 점수 송부
  - Institution Code `7713`
  - `TOEFL iBT / Paper Edition`만 가능
  - `Home Edition`, `ITP`, `MyBest` 불가
- TOEIC:
  - `공식 성적증명서(Official Score Certificate)` 원본 제출
  - `TOEIC IP` 불가

## 6. 선발 방식

건축학과는 다음 순서다.

1. 서류심사
2. 소논문(小論文)
3. 면접(面接)

## 7. 현재 최신 2027 일정표 기준 일정

- 출원 기간: `2026-08-25 ~ 2026-09-01`
- 1차 합격 발표: `2026-10-20`
- 2차 시험: `2026-11-07`
- 최종 합격 발표: `2026-12-18`
