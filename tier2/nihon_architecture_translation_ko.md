# 니혼대 건축계열 모집요강 번역 정리본

대상 원문:

- `2026년도 니혼대 외국인 유학생 선발 모집요강`
- 생산공학부 외국인 유학생 선발 안내 페이지

주의:

- 니혼대는 건축 관련 모집 단위가 여러 개라, 학교 전체보다 `지원 학부 단위`로 읽어야 한다.
- 이 문서는 공개 자료에서 확인이 비교적 선명한 `이공학부 건축학과`와 `생산공학부 건축공학과` 중심 번역 정리본이다.

## 1. 학교 전체 구조

공식 요강에 따르면, 니혼대 학부 외국인 유학생 선발에는

- `본교시험`
- `일본유학시험(EJU) 활용형`

두 방식이 있고, 학부별로 적용 방식이 다르다.

건축계열 확인 단위:

- 이공학부 건축학과
- 생산공학부 건축공학과
- 공학부 건축학과

## 2. 이공학부 건축학과

원문 핵심:

- `제2기 EJU 활용형`
- 일본어
- 수학 `코스2`
- 이과 `물리 필수 + 다른 1과목`
- `면접시험`
- `학과에 따라 구두시험을 포함할 수 있음`

EJU 활용 회차:

- `2024년 11월`
- `2025년 6월`

## 3. 생산공학부 건축공학과

### 제1기

- `본교시험`
- 기초학력검사: `영어 + 수학`
- 일본어 작문: `600자 이내`
- 면접

일정:

- 출원 마감: `2025-09-12`
- 시험: `2025-10-11`
- 합격 발표: `2025-10-23`

### 제2기

- `EJU 활용형`
- 일본어
- 수학 `코스2`
- 이과 `물리 필수 + 다른 1과목`
- 면접

일정:

- 출원 마감: `2026-01-23`
- 시험: `2026-02-20`
- 합격 발표: `2026-02-27`

## 4. 영어 요건

- 이공학부 건축학과: 별도 TOEFL/TOEIC 제출형으로 보이지 않음
- 생산공학부 건축공학과:
  - 제1기에서 `영어 시험` 직접 실시
  - 제2기는 EJU 활용형

즉:

- 니혼대 건축계열은 학교 전체가 하나의 영어 규칙을 갖는 것이 아니라, 학부별로 다르다.

## 5. 추가 준비 항목

- 생산공학부 제1기: `일본어 작문` 준비 필요
- 공식 확인 범위에서는 건축계열의 `지원이유서`나 `포트폴리오`는 직접 확인되지 않았다.

## 6. 한줄 해석

- 니혼대 건축계열은 `학부별로 구조가 다르다`는 점이 핵심이다.
- 특히 생산공학부는 제1기와 제2기 성격이 크게 다르다.
