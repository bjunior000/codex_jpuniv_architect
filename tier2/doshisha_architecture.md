# 도시샤대 건축계열 판정 메모

## 결론

- 2026-03-24 기준, 도시샤대 학부에는 `건축학과/건축계열` 모집 단위가 확인되지 않는다.
- 따라서 이번 프로젝트의 `건축학과 외국인 유학생 모집요강 정리` 대상에는 직접 해당하지 않는다.

## 근거

### 1. 학부 목록 공식 페이지

도시샤대 공식 학부 목록에는 다음과 같은 학부가 있다.

- 신학부
- 문학부
- 사회학부
- 법학부
- 경제학부
- 상학부
- 정책학부
- 문화정보학부
- 이공학부
- 생명의과학부
- 스포츠건강과학부
- 심리학부
- 글로벌커뮤니케이션학부
- 글로벌지역문화학부

여기에는 `건축학부` 또는 `건축학과`가 없다.

### 2. 실무 해석

- 도시샤대는 학부 기준으로 건축 전공을 직접 운영하는 대학으로 보기는 어렵다.
- 따라서 `건축 직접 비교표`에서는 제외하는 편이 맞다.
