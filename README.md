정리 원칙:

- 공식 사이트와 공식 PDF를 우선 사용
- 최신 연도 자료가 없으면 최신 공식 자료를 사용하고 그 사실을 명시
- PDF가 확보된 경우 `translation_ko.md` 형식의 번역 정리본 추가
- 학부 기준 `건축학과/건축계열 모집 단위가 없는 학교`는 그 사실과 공식 근거를 파일로 남김
- 일본어 또는 한자만으로 된 용어는 가능한 한 `한국어 명칭(일본어 원문)` 형식으로 병기
- 병기가 불필요한 일반 용어는 한국어로만 표기
